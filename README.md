# 🚀 IMORTAL-1 - Sistema de Bordo para Monitoramento de Sensores

## 📋 Sobre o Projeto

O **IMORTAL-1** é um sistema desenvolvido em linguagem **C** para simular o monitoramento de leituras de sensores em um sistema de bordo. O programa permite analisar dados coletados, calcular estatísticas básicas e gerar relatórios para auxiliar na interpretação das informações.

Este projeto foi desenvolvido com foco na prática de conceitos fundamentais de programação estruturada, modularização e manipulação de dados.

---

## ⚙️ Funcionalidades

- Cálculo da média das leituras;
- Identificação do valor máximo e mínimo;
- Cálculo dos desvios em relação à média;
- Verificação de leituras dentro de uma faixa aceitável;
- Exibição de barra gráfica simples em terminal;
- Geração de relatório completo;
- Simulação contínua com novas leituras.

---

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Compilador GCC
- Terminal/Console
